# ContactSection Component

A modern, professional, and fully responsive **Contact Section** for React portfolio websites, built using **React.js**, **Framer Motion**, and **Tailwind CSS**. Features animated backgrounds, floating icons, and a two-column layout with contact info and a contact form.  

---

## Features

- **React.js Component:** Easily reusable in any React project.  
- **Framer Motion Animations:** Smooth fade-ins, sliding effects, and floating animations.  
- **Two-Column Layout:**  
  - **Left:** Contact information + social links  
  - **Right:** Contact form with themed header  
- **Animated Background:** Floating circles and subtle moving shapes for a premium feel.  
- **Responsive Design:** Works perfectly on mobile and desktop screens.  
- **Themed Design:** Uses `primary` color for highlights and headers.  
- **Interactive Form:** Includes hover effects, focus states, and transitions for inputs and buttons.
